# Technical Decisions

## Decisión 001 — Stack principal

Fecha:

Pendiente.

Decisión:

Usar Next.js + React + TypeScript + React Three Fiber + Drei + Tailwind CSS.

Motivo:

Permite crear una experiencia 3D web moderna, modular, escalable y publicable fácilmente.

Alternativas consideradas:

* Spline embebido.
* Sitio estático.
* Unity WebGL.
* Webflow con embeds.

Consecuencia:

Mayor control y mejor integración, pero requiere cuidar performance, optimización de modelos y compatibilidad móvil.
