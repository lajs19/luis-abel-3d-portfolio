# Design System

## Personalidad visual

Profesional, técnico, moderno, limpio y confiable.

## Colores sugeridos

Azul marino:

Para identidad principal, paneles y detalles técnicos.

Blanco:

Para lectura clara y contraste.

Gris claro:

Para superficies, fondos secundarios y ambiente de oficina.

Azul eléctrico o cian técnico:

Para acentos interactivos, hover y elementos seleccionados.

## Estilo 3D

* Oficina técnica elegante.
* Taller CAD.
* Laboratorio de ingeniería.
* Modelos limpios.
* Escena ordenada.
* Elementos reconocibles.
* Evitar saturación visual.

## Tipografía

Preferir tipografías sans-serif modernas.

Opciones:

* Inter.
* Manrope.
* Montserrat.
* Geist.

## UI

* Paneles flotantes.
* Botones claros.
* Tarjetas con borde suave.
* Íconos técnicos.
* Animaciones sutiles.
* Hover visible.
* Estados de carga claros.

## Principio principal

El diseño debe impresionar, pero no confundir.

La experiencia debe sentirse profesional antes que “juego”.
