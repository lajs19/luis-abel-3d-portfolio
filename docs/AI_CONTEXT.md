## Objetivo del proyecto

Crear un CV/portafolio interactivo en 3D que funcione como una experiencia web inmersiva.

El usuario no quiere un currículum convencional ni una página web estática. Quiere un sitio que represente una oficina, casa/taller o laboratorio donde el visitante pueda explorar objetos y descubrir información profesional.

## Narrativa de la experiencia

El visitante entra a un espacio 3D. En lugar de leer una página tradicional, explora una oficina/taller.

Cada objeto importante abre una sección del perfil profesional.

Ejemplos:

* Una computadora muestra software CAD.
* Una torre eléctrica muestra experiencia en líneas de transmisión.
* Una maqueta de fundación muestra experiencia en fundaciones.
* Un panel de proyectos muestra proyectos relevantes.
* Una pantalla de simulación muestra FEA/CFD.
* Una biblioteca muestra normas.
* Un diploma muestra certificaciones.
* Una tarjeta profesional muestra contacto.
