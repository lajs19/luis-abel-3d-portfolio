# Roadmap

## Semana 1

Objetivo:

Crear base documental y técnica.

Entregables:

* Repositorio creado.
* Documentación inicial lista.
* Proyecto Next.js inicial.
* Estructura de carpetas.
* Datos base.

## Semana 2

Objetivo:

Crear MVP funcional.

Entregables:

* Escena 3D simple.
* Objetos clicables placeholder.
* Paneles informativos.
* Descarga de CV.
* Contacto.

## Semana 3

Objetivo:

Mejorar escena y contenido.

Entregables:

* Diseño visual mejorado.
* Proyectos destacados.
* Objetos representativos.
* Mejor UX.
* Primera revisión QA.

## Semana 4

Objetivo:

Optimizar y publicar.

Entregables:

* Optimización de rendimiento.
* Responsive.
* Build estable.
* Deploy en Vercel o Netlify.
* Revisión final.
