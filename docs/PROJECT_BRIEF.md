# Project Brief

## Nombre del proyecto

Luis Abel 3D Portfolio

## Nombre conceptual

Luis Abel Design Lab

## Descripción corta

CV y portafolio interactivo en 3D presentado como una oficina técnica/taller de ingeniería donde el visitante explora objetos para descubrir información profesional.

## Usuario objetivo

* Reclutadores.
* Clientes potenciales.
* Empresas de diseño, ingeniería, arquitectura, energía, construcción e industria.
* Directores de proyectos.
* Encargados de contratación técnica.

## Objetivo profesional

Posicionar a Luis Abel como un perfil técnico confiable para trabajos de:

* Diseño de proyectos.
* Supervisión de diseño.
* Coordinación técnica.
* Diseño mecánico.
* Diseño 3D.
* Líneas de transmisión eléctrica.
* Fundaciones.
* Documentación técnica.
* Proyectos industriales, civiles y energéticos.

## Problema que resuelve

Un CV convencional no muestra adecuadamente las capacidades de diseño, visualización 3D, supervisión técnica y pensamiento espacial de Luis Abel.

Este portafolio convierte su perfil en una experiencia visual e interactiva.

## Resultado esperado

Un sitio web 3D funcional, profesional, rápido y claro, que permita:

* Explorar el perfil profesional.
* Ver habilidades técnicas.
* Ver proyectos.
* Descargar CV.
* Contactar a Luis Abel.
